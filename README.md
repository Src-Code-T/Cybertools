## 📌 Overview

**Cybertools** is a curated repository of Python-based utilities designed to explore core concepts in cybersecurity such as:

* System monitoring
* Input capture mechanisms
* Networking fundamentals
* OS-level interactions

This repository is intended to support **hands-on learning** and **deep technical understanding** of security-related systems.

## 🔍 Tools Included

### ⌨️ Keylogger V1

**Description:**
A basic keylogger implementation to understand how keyboard input can be captured at the OS level.

**Concepts Covered:**

* Keyboard event listeners
* Logging mechanisms
* File handling

**Usage:**

```bash
cd Keylogger_V1
python keylogger.py
```

**Note:**
Run only in controlled environments.
